# Informal-Manufacturing-Taxation-Project
This project analyzes informal manufacturing taxation through spatial data visualization. It aims to provide insights into the distribution of informal manufacturers and their taxation relationships using heatmaps.
## Requirements
- Python 3.9 or higher
- Required libraries: 
  - `pandas`
  - `matplotlib`
  - `seaborn`
  - `a csv file with point locations of informal manufacturers was used for this specifically`

## How to Run
1. Clone the repository.
2. Install the requirements using pip
3. Run the spatial analysis script
